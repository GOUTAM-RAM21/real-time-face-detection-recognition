This project implements a real-time face recognition system using Python, OpenCV, and dlib.
It detects faces from a webcam feed, encodes known faces, and recognizes them in real time.

Features:

✅ Real-time face detection using webcam

✅ Face recognition with pre-stored images

✅ Face encoding and comparison

✅ Image capture for new faces

✅ High accuracy using dlib face encodings
